🎬 Proyecto: Clasificación de Reseñas de Películas (Film Junky Union)
🔧 Elementos Principales
📊 Datos
- Dataset IMDB (imdb_reviews.tsv) con reseñas de películas
- Campos principales:
- Texto de la reseña
- Polaridad (positiva / negativa)
- Conjunto (entrenamiento / prueba)
💻 Herramientas Técnicas
- Jupyter Notebook — Documentación completa del proceso
- Vectorización de texto (TF‑IDF) — Conversión de texto a representaciones numéricas
- BERT (opcional) — Embeddings avanzados para mejorar el desempeño
- Modelos de Machine Learning — Regresión logística, gradient boosting, entre otros

🎯 Estructura del Proyecto (9 Pasos)
<br>1️⃣ Carga de Datos
- Importación del dataset de reseñas IMDB
<br>2️⃣ Preprocesamiento
- Limpieza y normalización del texto
- Eliminación de ruido y caracteres irrelevantes
<br>3️⃣ Análisis Exploratorio
- Revisión del balance entre clases
- Exploración de longitud de reseñas y patrones comunes
<br>4️⃣ Preprocesamiento para ML
- Vectorización de textos con TF‑IDF
- Preparación de matrices de entrenamiento y prueba
<br>5️⃣ Entrenamiento
- Entrenamiento de mínimo 3 modelos diferentes
- Ajuste de hiperparámetros
<br>6️⃣ Evaluación
- Medición del desempeño en el conjunto de prueba
- Métricas: accuracy, F1‑score, matriz de confusión
<br>7️⃣ Pruebas Personalizadas
- Clasificación de reseñas creadas por el usuario
<br>8️⃣ Comparación
- Análisis de diferencias entre modelos
- Identificación del mejor enfoque
<br>9️⃣ Conclusiones
- Resumen de hallazgos
- Limitaciones y oportunidades de mejora
